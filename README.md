**Step 1 — Record your screen**
- Press **Windows + G** → click record button
- Or use [OBS Studio](https://obsproject.com) (free)

1. Go to [youtube.com](https://youtube.com) → click **+** → Upload video
2. Set visibility to **"Unlisted"** ← people need the link but can't find it by search
3. After upload, copy your video URL — e.g. `youtube.com/watch?v=ABC123`
4. The ID is the part after `v=` → `ABC123`
